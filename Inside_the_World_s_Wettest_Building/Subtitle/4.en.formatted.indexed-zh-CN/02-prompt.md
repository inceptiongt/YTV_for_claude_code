# Translation Prompt

## Task
将以下 YouTube 视频字幕从英文翻译为简体中文（zh-CN）。

## Style & Audience
- 风格：conversational（对话式）—— 像朋友讲述有趣见闻，生动、自然、不生硬
- 受众：普通大众，对工程/科学感兴趣但无专业背景
- 语气：与原文保持一致——主持人风格轻松幽默；研究员发言用自然中文意译

## Format Rules（必须严格遵守）
1. **保留每行 [N] 序号**，与原文一一对应，不合并、不拆分行
2. **翻译 # 章节标题**
3. 不增加、不删除任何 [N] 编号行
4. 数字、单位原样保留（8 mm、300 mm/h、75 km 等）
5. 专业术语首次出现时可加原文括注

## Special Cases
- 日本研究员的英文发言（语法不规范）：理解原意，用自然流畅的中文表达，不翻译语法错误
- [snorts] 等音效：保留括号，翻译为 [轻哼]
- >> 符号：原样保留（表示不同说话人）
- 幽默和自嘲语气：用口语化中文再现，保持轻松感
