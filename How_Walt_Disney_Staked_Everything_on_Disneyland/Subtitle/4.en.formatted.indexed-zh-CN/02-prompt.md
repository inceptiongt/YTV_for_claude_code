# Translation Context

## Task
Translate a Disneyland documentary subtitle file (EN → zh-CN). Style: conversational (friendly, spoken-like, approachable). Audience: general viewers.

## Critical Format Rules (MUST follow)
- Every `[N]` line must appear exactly once in output — same count, same order, no merging, no splitting
- Translate `# Chapter Heading` lines too
- Preserve all `>>` markers (indicate speaker/interview)
- Output format identical to input

## Terminology
- Disneyland → 迪士尼乐园
- Walt Disney World → 华特迪士尼世界
- Imagineer → 幻想工程师（首次出现时注明：幻想工程师，即 Imagineer，迪士尼对"想象力+工程师"的创意融合称谓）
- Forced perspective → 强迫透视法
- Weenies → 视觉诱饵（Disney的术语，指引导游客视线的标志性景观）
- Go Away Green → "消隐绿"（Disney乐园专用涂料，让设施融入绿植背景）
- Main Street USA → 美国大街
- Sleeping Beauty Castle → 睡美人城堡
- Jungle Cruise → 丛林巡游
- Mark Twain (steamboat) → 马克·吐温号
- Herb Ryman → 赫伯·赖曼
- Harper Goff → 哈珀·戈夫
- Cornelius Vanderbilt Wood → 科尼利厄斯·范德比尔特·伍德
- Brick Borrow → Brick Borrow（品牌名保留）
- "silvercreen" (typo for silver screen) → 银幕
- "bolard" (typo for bollard) → 护柱

## Style Notes
- Short sentences should stay punchy in Chinese
- Rhetorical questions should feel natural spoken
- Documentary narration: informative but engaging, not stiff
