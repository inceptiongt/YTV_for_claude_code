# Translation Prompt

## Task

## Critical Format Rules
1. Every `[N]` numbered line in the source → exactly one `[N]` line in output. No merging, no splitting.
2. Preserve `[N]` prefixes exactly as-is.
3. Translate all `# Heading` lines.
4. Preserve `>>` markers at the start of lines exactly as-is.
5. Keep blank lines between sections.
